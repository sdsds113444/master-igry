# 🚀 Как выложить сайт «Мастер игры» в интернет (бесплатно)

Инструкция для новичка. Ничего платить не нужно.
Сейчас сайт работает на **мок-данных** (демо) — его уже можно выложить и показать.
База данных (Supabase) подключается позже, когда утвердим демо.

---

## Часть A. Выложить визуал на Vercel (5–10 минут, бесплатно)

### Шаг 1. Аккаунт GitHub
1. Зайди на **github.com** → «Sign up» → заведи бесплатный аккаунт.

### Шаг 2. Залить код в GitHub
В терминале из папки проекта (`master-igry`) выполни по очереди:

```bash
git init
git add .
git commit -m "Мастер игры — MVP"
```

Затем на github.com нажми **New repository** → имя `master-igry` → **Create**.
GitHub покажет команды — скопируй две строки вида:

```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/master-igry.git
git push -u origin main
```

> Если попросит логин/пароль — вместо пароля нужен **токен** (GitHub → Settings → Developer settings → Personal access tokens). Скажи мне — проведу за руку.

### Шаг 3. Деплой на Vercel
1. Зайди на **vercel.com** → «Sign up» → войти **через GitHub**.
2. «Add New…» → **Project** → выбери репозиторий `master-igry` → **Import**.
3. Vercel сам поймёт, что это Vite. Ничего не меняй → **Deploy**.
4. Через ~1 минуту получишь ссылку вида **`master-igry.vercel.app`** — это твой живой сайт. 🎉

> Каждый раз, когда зальёшь новый код в GitHub (`git push`), Vercel **сам** пересоберёт сайт.

---

## Часть B. Подключить базу Supabase (когда утвердим демо, бесплатно)

Нужна для: входа команд, сдачи ответов, хранения баллов и рейтинга.

1. Зайди на **supabase.com** → «New project» (бесплатный tier).
2. Открой **Project Settings → API** и скопируй два значения:
   - `Project URL`
   - `anon public` ключ
3. **Пришли их мне** — я подключу базу к сайту и заведу таблицы
   (команды, участники, задания, ответы, баллы) и вход по коду команды.

Бесплатных лимитов Supabase + Vercel **хватает на 300–450 человек** с запасом.

## Как запустить у себя на компьютере
```bash
npm install      # один раз
npm run dev      # открыть http://localhost:5173
```
Демо-вход: любой код, например **KOYA-04**.
